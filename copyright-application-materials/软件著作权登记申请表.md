## 二、软件功能简介

- 微信登录、注册资料完善、角色切换与用户信息维护。
- 学生端支持加入班级、查看训练任务、提交作业、查看提交记录和排行榜。
- 教师端支持创建和管理班级、发布训练任务、查看学生提交并进行审核评分。
- 任务体系支持公开任务和班级任务，包含项目分类、难度、积分、截止时间和素材附件。
- 审核体系支持入班审批、作业审核、反馈记录、反馈附件、积分发放和操作追踪。
- 排行榜体系支持周榜、月榜、总榜以及后台排行数据查询。
- 后台管理网站支持运营总览、系统配置、项目配置、用户、班级、任务、提交、公告和日志管理。
- 基于腾讯云开发 CloudBase，使用云函数、文档数据库和云存储承载后端能力。

## 三、软件技术特点

- 采用微信小程序原生框架实现学生端和教师端，使用自定义 TabBar、页面路由、组件化页面和 TDesign Mini 组件库。
- 后端基于腾讯云开发 CloudBase，云函数按业务拆分为用户、班级、任务、提交、审核、排行榜、公告和后台管理等能力。
- 后台管理端采用 Vue 3、Vite、Pinia、Vue Router、TDesign Vue Next 和 ECharts，提供运营与管理控制台。
- 通过统一 API 封装、服务层封装、共享云函数模块和权限校验逻辑降低业务耦合。
- 数据存储采用 CloudBase 文档数据库，核心集合包括：users, classes, class_memberships, class_join_applications, tasks, submissions, projects, ranking_snapshots, system_config, announcements, announcement_reads, operation_logs。
- 上传资料、任务素材和反馈附件通过 CloudBase 云存储管理，并在小程序端做数量、类型和大小约束。
