# 智力控码软件 用户手册

著作权人名称：李崇伟

## 一、软件简介

智力控码软件是一套基于微信小程序和 CloudBase 的教育培训管理软件，面向编程、无人机、机器人等训练项目，提供学生学习任务、教师教学管理、提交审核、积分排行和后台运营管理功能。

### 1.1 软件概述

软件包含微信小程序端和后台管理网站。小程序端面向学生和教师，学生可注册登录、加入班级、查看任务、提交作品、查看积分与排行榜；教师可管理班级、发布任务、审核提交、处理入班申请。后台管理网站面向管理员，提供运营统计、系统配置、项目配置、用户、班级、任务、提交、排行、公告和日志管理。

### 1.2 主要特点

- 微信登录、注册资料完善、角色切换与用户信息维护。
- 学生端支持加入班级、查看训练任务、提交作业、查看提交记录和排行榜。
- 教师端支持创建和管理班级、发布训练任务、查看学生提交并进行审核评分。
- 任务体系支持公开任务和班级任务，包含项目分类、难度、积分、截止时间和素材附件。
- 审核体系支持入班审批、作业审核、反馈记录、反馈附件、积分发放和操作追踪。
- 排行榜体系支持周榜、月榜、总榜以及后台排行数据查询。
- 后台管理网站支持运营总览、系统配置、项目配置、用户、班级、任务、提交、公告和日志管理。
- 基于腾讯云开发 CloudBase，使用云函数、文档数据库和云存储承载后端能力。

## 二、功能概述

### 2.1 功能列表

1. 用户登录与注册：用户通过微信身份进入小程序，完成姓名、手机号、学校、年级等资料登记。
2. 角色管理：同一用户可拥有学生、教师、管理员等角色，并在授权范围内切换使用视图。
3. 班级管理：教师创建班级、维护班级信息、分享邀请码；学生可申请加入班级并查看班级详情。
4. 任务管理：教师发布公开任务或班级任务，配置项目、分类、难度、积分、截止时间、任务素材和附件。
5. 作业提交：学生查看任务详情后上传图片或文件，填写说明并提交作业，可查看提交记录。
6. 审核评分：教师查看待审核提交，填写审核结果、反馈说明、反馈附件，并根据规则发放积分。
7. 积分排行：系统根据积分和完成情况生成排行，学生可查看周榜、月榜和总榜。
8. 公告通知：管理员可维护公告，小程序端可展示通知列表和公告内容。
9. 后台管理：管理员通过 Web 控制台查看运营数据、管理配置、项目、用户、班级、任务、提交、排行和日志。

### 2.2 适用场景

- 青少年编程、无人机、机器人课程的课后训练管理。
- 教师发布训练任务并收集学生作品。
- 机构运营人员查看训练数据、配置项目参数和维护系统公告。
- 需要积分激励、排行榜和提交审核机制的教育训练场景。

## 三、安装/使用说明

### 3.1 系统要求

- 小程序端：微信客户端，支持微信小程序基础库 3.0 及以上。
- 后台管理端：现代浏览器，支持 HTTPS 访问 CloudBase 静态网站托管地址。
- 网络要求：需保持互联网连接以访问云函数、云数据库和云存储。

### 3.2 启动软件

1. 用户打开微信，搜索或扫描“智力控码”小程序二维码进入。
2. 首次进入时根据页面提示完成微信登录和资料注册。
3. 管理员访问后台管理网站地址，完成登录和管理员权限校验后进入控制台。

## 四、主要功能说明

### 4.1 学生端使用

1. 进入小程序后完成登录和资料注册。
2. 在首页查看个人训练概览、任务入口和公告信息。
3. 在班级页面输入邀请码或通过分享链接申请加入班级。
4. 在任务中心筛选公开任务或班级任务，查看任务详情。
5. 在任务提交页上传图片或附件并填写作品说明。
6. 在提交记录中查看审核状态、教师反馈和积分结果。
7. 在排行榜页面查看个人排名和榜单数据。

### 4.2 教师端使用

1. 在个人中心切换到教师角色。
2. 进入班级管理页面，新建班级或维护已有班级。
3. 查看学生入班申请，执行同意或拒绝操作。
4. 进入任务管理页面，新建、编辑、发布或关闭训练任务。
5. 在审核页面查看学生提交记录，填写审核结论、评分或反馈信息。
6. 查看班级详情和任务统计，跟踪学生训练进度。

### 4.3 后台管理端使用

1. 打开后台管理网站并完成管理员登录。
2. 在运营概览查看用户、班级、任务、提交和审核等关键指标。
3. 在系统配置页面维护注册积分、每日积分上限、任务提交次数、抽奖开关等参数。
4. 在项目配置页面维护编程、无人机、机器人等训练项目、分类和难度。
5. 在用户、班级、任务、提交、排行、公告和日志页面进行查询和管理。
6. 对关键写操作进行审计追踪，便于运营复盘和问题定位。

## 五、注意事项

- 请确认网络连接正常，云函数、数据库和云存储服务可访问。
- 用户首次使用需完成微信授权和资料注册，否则部分业务功能不可用。
- 教师和管理员功能需要对应角色权限，未授权用户无法访问管理能力。
- 上传图片和文件需符合系统设定的数量、类型和大小限制。
- 正式运营前应核对 CloudBase 环境、数据库集合、安全规则和后台管理员绑定关系。

## 六、常见问题

### 6.1 无法进入教师功能怎么办？

请确认当前账号在用户资料中具备教师角色，并在个人中心切换到教师视图。

### 6.2 提交任务失败怎么办？

请检查任务是否仍在开放状态、是否超过提交次数、文件是否超出大小限制，并确认网络连接正常。

### 6.3 后台无法登录怎么办？

请确认后台账号已绑定管理员身份，`users` 集合中包含管理员角色和有效的 `admin_auth_uid`。
