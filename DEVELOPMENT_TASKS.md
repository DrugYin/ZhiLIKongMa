## 项目开发总览

### 技术栈说明

| 类别 | 技术 | 说明 |
|------|------|------|
| 前端框架 | 微信小程序 | 基础库 3.0+ |
| 开发语言 | JavaScript (ES6+) | 不使用 TypeScript |
| UI 组件库 | TDesign Mini | 1.x 版本 |
| 后端服务 | 微信云开发 + CloudBase | 云函数/数据库/存储 |
| 样式 | WXSS | 直接使用，非 Less |

### 项目结构说明

```
ZhiLiKongMa/
├── assets/              # 静态资源（头像等）
├── cloudfunctions/      # 云函数目录
├── components/          # 公共组件
│   ├── custom-navbar/   # 自定义导航栏
│   ├── empty/           # 空状态组件
│   ├── form/            # 表单组件
│   └── loading/         # 加载组件
├── config/              # 配置文件
│   └── project.js       # 项目配置服务
├── custom-tab-bar/      # 自定义 TabBar
├── miniprogram_npm/     # NPM 构建产物
├── pages/
│   ├── login/           # 登录页
│   ├── student/         # 学生端页面
│   │   ├── index.js     # 首页
│   │   ├── mine/        # 我的
│   │   ├── rank/        # 排行榜
│   │   ├── setting/     # 设置
│   │   └── training/    # 训练
│   └── teacher/         # 教师端页面
│       ├── index.js     # 首页
│       ├── class-manage/ # 班级管理
│       ├── task-manage/ # 任务管理
│       ├── mine/        # 我的
│       └── pending/     # 审核
├── services/            # 服务层
│   ├── api.js           # API 统一调用
│   ├── auth.js          # 认证服务
│   └── storage.js       # 存储服务

    ├── constant.js      # 常量定义
```

## 阶段二：用户系统开发

### 阶段目标
实现统一用户系统的注册、登录、角色切换和个人信息管理功能。所有用户通过微信授权登录，使用统一的 users 表，通过 role 字段控制角色权限。

### 预计工期
**1.5 周**

### 核心设计

```
┌─────────────────────────────────────────────────────────────────┐
│                      用户系统设计                                │

│  │  avatar_url: "cloud://xxx"              │                    │
│  │  phone: "13800138000"                   │                    │

│  角色切换：                                                      │
│  - 学生端 TabBar: 首页、排行、我的                               │
│  - 教师端 TabBar: 首页、审核、班级、我的                         │
│                                                                 │

```

```javascript
// custom-tab-bar/index.js

学生端 TabBar:
- 首页 (home)
- 排行 (leaderboard)
- 我的 (user)

教师端 TabBar:
- 首页 (home)
- 审核 (pen-ball)
- 班级 (work)

切换逻辑：
1. 根据 current_role 判断
2. 调用 changeData(type) 更新 TabBar
3. 使用 wx.switchTab 切换页面
```
